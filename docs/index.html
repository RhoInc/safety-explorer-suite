Overview

The Safety Explorer Suite consists 6 custom interactive graphics and 2 helper libraries. The 6 graphics libraries improve upon traditional static reports by adding simple user interface elements, including filters, dynamic searches, toggles for display options, and linked charts and listings. This page includes links to examples, the 8 libraries and detailed usage instructions providing an overview of how to get started with the tools.
